# DreamLauncher

DreamLauncher — лаунчер проекта **DreamWars**.
## Возможности

- Запуск Minecraft версий **1.6.4 – 1.21.1**
- Загрузка и обновление файлов с сервера
- Проверка клиентских файлов на изменения
- Защита процесса от инжектов и стороннего ПО
- Античит-модули на **JNA + JNI**
- Настройки запуска:
  - выделение оперативной памяти
  - полноэкранный режим
  - параметры JVM

## Технический стек

- Java 8  
- JavaFX  
- Netty  
- JNA + JNI  
- Gradle 8.14  
- Windows only  

## О проекте

Лаунчер создан для полного контроля над клиентом.  
Без модификаций. Без читов. Без лишнего.
